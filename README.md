# Emotion Regulation Music Recommender

## Description
This project analyzes user emotions from text input and recommends music using Spotify API to regulate mood.

## Technologies Used
- Python
- Streamlit
- NLP (Text Emotion Detection)
- Spotify Web API

## Features
- Emotion detection
- Mood-based music recommendation
- Interactive UI
- Real-time Spotify search

## How to Run
streamlit run app.py
